## Project Description
social is a multi-page React application built using React Router and API integration. It demonstrates advanced 
React concepts such as hooks, data fetching, routing, and reusable components.

## Technologies Used
- React
- React Router DOM
- JavaScript (ES6+)
- CSS

## Features
- Multi-page routing (Home, Posts, About)
- Dynamic post details page
- API data fetching from JSONPlaceholder
- Loading states
- Error handling
- Custom hooks (useFetch)
- Responsive UI

## How to Run
1. Clone repository
2. Run `npm install`
3. Run `npm run dev`

## Lessons Learned
- React Router setup and navigation
- useEffect for data fetching
- Custom hooks for reusable logic
- Handling API loading and errors

## Challenges Faced
- Managing asynchronous API calls
- Setting up dynamic routing
- Handling loading states properly
